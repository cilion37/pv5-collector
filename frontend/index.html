⚙️ 초기 설정 가이드

1
YouTube Data API v3 키 발급
Google Cloud Console → 새 프로젝트 → API 및 서비스 → YouTube Data API v3 활성화 → 사용자 인증 정보 → API 키 생성
2
Google Custom Search 설정
Programmable Search Engine → 새 검색엔진 생성 (전체 웹 검색) → 검색엔진 ID(cx) 복사
Cloud Console에서 Custom Search JSON API도 활성화 후 API 키 복사
3
우측 상단 ⚙ 설정 버튼 클릭 → API 키 입력 → 저장
4
지금 수집 버튼 클릭하면 즉시 수집 시작! 매일 오전 9시 자동 수집됩니다.
※ API 키는 브라우저 로컬에 저장됩니다. Worker 배포 후에는 wrangler secret으로 서버에 설정하세요.
※ YouTube API: 무료 1일 10,000 쿼터 / Google Custom Search: 무료 100회/일
